oh-my-vim
===============

My personal VIM setting.  
I use it to write:  
	
	1) php
	2) c/c++
	3) lua
	4) markdown

#Installation

1. clone files:
	
		git clone https://github.com/pangliang/oh-my-vim.git ~/.vim 

2. run `update.sh` to all modules:
	
		cd ~/.vim
		./init.sh

3. link `~/.vimrc` to `oh-my-vim/vimrc`

		ln -s .vim/vimrc .vimrc
